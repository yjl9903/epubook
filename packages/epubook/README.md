# epubook

A Node EPUB generation library.

It will support the generation of the [latest epub standard](https://www.w3.org/TR/epub-33/) (3.3).

## Installation

```bash

```

## Usage

```ts
import { Epubook } from 'epubook'

await Epubook.create({ title: 'test' })
```
